Télécharger et installer le drivers USB pour Android

La synchronisation a connexion directe USB driver nécessite la configuration initiale des les Android USB drivers pour Windows, s'il ne sont pas déjà installé sur votre ordinateur.

Les drivers USB pour Windows sont spécifiques à chaque fabricant de périphérique Android. Ils sont disponibles gratuitement à partir des sites Web des fabricants. Votre ordinateur peut déjà avoir l'Android USB drivers installé si vous avez installé un logiciel de synchronisation, de sauvegarde/backup ou similaire qui se connecte à votre dispositif via le port USB.

USB drivers pour Android sont aussi connus comme ADB (Android Debug Bridge). Ils sont disponibles comme téléchargement gratuit à partir des fabricants (OEM's).

Le dispositif Android doit avoir le mode Débogage USB activé avant d'installer les drivers et avant d'utiliser la synchronisation a Connexion directe USB driver:
 
Sur périphériques Android 1.x à 3.x allez dans Paramètres - Applications - Développement - activer Débogage USB
 
Sur périphériques Android 4.0 et 4.1 allez dans Paramètres - Options pour les développeurs - Activer le Débogage USB
 
Sur périphériques Android 4.2 et périphériques plus récents allez dans Paramètres > Général (Plus) > Options de développement > Activer le Débogage USB
 
NOTE: Sur périphériques Android 4.2 et périphériques plus récents (i.e Samsung Galaxy Note 3 et S4), vous devrez peut-être débloquer "Options de développement" avant qu'il soit disponible dans le menu:
- Allez dans Settings > Général (Plus)
- Appuyez "À propos du périphérique"
- Appuyez sur le bouton "Numéro de version" 7 fois.
"Options de développement" devrait maintenant être déverrouillé et disponible dans Paramètres > Général (Plus) > Options de développement
