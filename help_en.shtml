Online Help - Synchronize Notes

The Journal Notes Multimedia Windows software can synchronize with its companion Android and BlackBerry applications. The Android app is available for download on Google Play. A BlackBerry version is available on BlackBerry World.

Get the Journal Notes Multimedia Android app on Google Play

Get the Journal Notes Multimedia BlackBerry app on BlackBerry World

Android Sync

Journal Notes Multimedia will synchronize with Android phones and tablets via WiFi (wireless) network or USB cable. No Internet access is required.

WiFi synchronization works when both the computer and the Android device are connected to same wireless network (to the same network router).

Synchronization via USB cable can be done in two ways: USB driver direct sync and USB storage transfer sync. The USB driver direct sync is easier to use but it requires the initial configuration of the Android USB drivers for Windows, if they are not already installed on your PC.

USB storage sync does not require any configuration but it works only on Android devices with an SD card or equivalent partition. This method is also slower to use as it requires the USB storage to be turned on and off each time a sync is started and during the synchronization.

Android WiFi sync

Connect the PC and the Android device to the same wireless network.

The sync mode is started by opening the WiFi sync screen (from the Options menu) on the device and the corresponding menu on the PC. The connection will always be started from the device FIRST.

Press the CONNECT button on the device BEFORE clicking the CONNECT button on the computer to establish the connection.

Once the WiFi connection is established, click the START SYNC button on the computer to synchronize the notes.

NOTE: Checking the "Remove all deleted notes" checkbox will permanently remove the deleted notes from all the devices and computers participating in the sync.

Android USB driver direct sync

The USB cable must be connected between the computer and the device.

The Android device must have the USB Debugging mode ENABLED before installing drivers and before using USB direct driver sync:
 
On Android 1.x to 3.x devices go to Settings > Applications > Development > Enable USB debugging
 
On Android 4.0 and 4.1 devices go to System Settings > Developer Options > Enable USB debugging
 
On Android 4.2 and newer devices go to Settings > General (More) > Developer Options > Enable USB debugging
 
For the Android 4.2 and newer devices (i.e Samsung Galaxy Note 3 and S4), you may need to unlock Developer Options before it is available within the Menu:
- Go to Settings > General (More)
- Tap "About Device"
- Tap on the "Build Number" button 7 times.
Developer Mode should now be unlocked and available in Settings > General (More) > Developer Options

The sync mode is started by opening the USB driver direct screen on the device and the corresponding menu on the PC. The connection will always be started from the device FIRST.

Press the CONNECT button on the device BEFORE clicking the CONNECT button on the PC to establish the connection.

Once the USB connection is established, click the START SYNC button on the computer to synchronize the notes.

NOTE: Checking the "Remove all deleted notes" checkbox will permanently remove the deleted notes from all the devices and computers participating in the sync.

Android USB driver direct sync initial configuration

This sync mode requires the initial configuration of the Android USB drivers for Windows, if they are not already installed on your PC.

The USB drivers for Windows are specific for each Android device manufacturer. They are available for free from the manufacturers web sites. Most likely, your computer has the drivers already installed.

Check this listing for Android USB driver download and install options.

Android USB storage transfer sync

USB storage sync works only on Android devices with an SD card or equivalent partition (mass storage). This method is slower to use as it requires the USB storage to be turned on and off each time a sync is started and during the synchronization.

The USB storage mode is used on Android phones and tablets to transfer files with a computer via USB cable. The USB storage setting is accessible after the USB cable is plugged, usually in the upper notification bar or the Settings screen. This mode must be toggled on and off while transferring notes because the computer and the device cannot access the SD Card at the same time.

Steps to sync new notes from the Android device to the PC:

  • connect the USB cable
  • turn the USB storage mode off
  • press the "Put Notes to PC" button on the device app
  • turn the USB storage mode on
  • click the "Get Notes from Device" button on the PC software

Steps to sync new notes from the PC to the Android device:

  • connect the USB cable
  • turn the USB storage on
  • click the "Put Notes to Device" button on the PC software
  • turn the USB storage off
  • press the "Get Notes from PC" button on the Android app

NOTE: only the new notes will be transferred and added to the existing ones, independently on each device participating in the sync. Existing notes will not be updated by the sync. To delete a note permanently it must be manually removed from all the devices participating in the sync.

BlackBerry BlackBerry Sync

The BlackBerry can be connected to a computer with a USB cable. Once connected, it will show as an external drive to the computer. The computer will use this drive to transfer the new notes with the BlackBerry.

The Blackberry drive requires the initial setting of the support software that comes with the device. This software is usually installed automatically when the BlackBerry is connected to the computer for the first time. Alternately, it can be downloaded from the BlackBerry support website.

Steps to sync new notes from the BlackBerry to the PC:

  • connect the USB cable and make sure the BlackBerry drive is showing in Windows Explorer
  • press the "Put notes to computer" button on the BlackBerry app
  • click the "Get new notes from BlackBerry" button on the computer software

Steps to sync new notes from the computer to the BlackBerry:

  • connect the USB cable and make sure the BlackBerry drive is showing in Windows Explorer
  • click the "Send new notes to BlackBerry" button on the computer software
  • press the "Get notes from computer" button on the BlackBerry app

NOTE: only the new notes will be transferred and added to the existing ones, independently on each device participating in the sync. Existing notes will not be updated by the sync. To delete a note permanently it must be manually removed from all the devices participating in the sync.
