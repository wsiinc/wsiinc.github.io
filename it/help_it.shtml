Aiuto - Sincronizzazione delle note

Il computer software Diario Note Multimedia può sincronizzare le note con le applicazioni di accompagnamento per Android e BlackBerry. La app per Android e scaricabile dal negozio Google Play. La app per BlackBerry e scaricabile dal negozio BlackBerry World

Clicca qui per scaricare Diario Note Multimedia dal BlackBerry World

Sincronizzazione Android

Diario Note Multimedia sincronizza con i telefoni e tablet Android attraverso la rete locale senza fili (WiFi) o il cavo USB. Non è richiesta alcuna connessione a Internet

La sincronizzazione WiFi funziona quando sia il computer e il dispositivo Android sono collegati alla stessa rete wireless (allo stesso router di rete).

La Sync via USB può essere fatta in due modi: Sync USB a connessione diretta e Sync a trasferimento Archivio di massa USB.

La modalità Sync USB connessione diretta è più facile da usare ma necessita la configurazione iniziale dei Android USB driver per Windows, se non sono già installati sul computer.

La modalità Sync transferimento a Archivio di massa USB non necessita nessuna configurazione ma funziona solo su dispositivi Android con una scheda SD o una partizione equivalente. Questo metodo è inoltre più lenta da utilizzare in quanto richiede attivare e disattivare l'Archivio di massa USB ogni volta una sincronizzazione e avviata e durante la sincronizzazione.

Android-Sync a rete WiFi locale (senza fili)

Collega il computer e il dispositivo Android alla stessa rete WiFi.

La sincronizzazione viene avviata aprendo la schermata WiFi sync (nel menu Opzioni) sul telefone/tablet e il menu corrispondente sul computer. Il collegamento verrà sempre avviato PRIMO dal dispositivo Android.

Volta scollegato, premi il pulsante RICOLLEGARE sul dispositivo Android prima di fare clic sul pulsante COLLEGA del computer per ristabilire la connessione.

Una volta la connessione WiFi è stabilita, clicca sul pulsante "Avviare la sincronizzazione" sul computer per sincronizzare le note.

NOTA: selezionando la casella "Togliere tutte le note cancellati" rimuovera definitivamente le note eliminati da tutti i dispositivi e computer che partecipano alla sincronizzazione.

Android Sync - cavo USB connessione diretta

Il cavo USB deve essere collegato prima di iniziare la sincronizazione.

La modalità Debug USB deve essere ATTIVATA prima di iniziare la sincronizazione:

Sui dispositivi Android 1.x - 3.x vai al Impostazioni > Applicazioni > Sviluppo > USB debugging attivato
 
Sui dispozitivi Android 4.0 e 4.1 vai al Impostazioni > USB debugging attivato
 
Sui dispozitivi Android 4.2 e successivi vai su Impostazioni > Generale (Più) > Opzioni Sviluppatore > USB debugging attivato
 
Sui dispozitivi Android 4.2 e successivi (i.e Samsung Galaxy Note 3 e S4), potrebbe essere necessario sbloccare le Opzioni Sviluppatore prima che sia disponibile all'interno del menu:
- Vai al Impostazioni > Generale (Più)
- Premi "Info sul dispozitivo"
- Premi il tasto "Numero Build" 7 volte.
Modo Sviluppo dovrebbe essere sbloccato e disponibile in Impostazioni > Generale (Più) > Opzioni Sviluppatore
La sicronizzazione e iniziata dalla schermata Sync USB connessione diretta sul dispositivo ed il menu corrispondente sul computer. La connessione deve essere iniziata dal dispositivo Android PRIMA.

Premi il pulsante COLLEGA sul dispositivo PRIMA di cliccare COLLEGA sul computer per iniziare la connessione.

Una volta la connessione USB è aperta, clicca sul pulsante "Avviare la sincronizzazione" sul computer per sincronizzare le note.

NOTA: selezionando la casella "Togliere tutte le note cancellati" rimuovera definitivamente le note eliminati da tutti i dispositivi e computer che partecipano alla sincronizzazione.

Android-Sync USB connessione diretta-configurazione iniziale

Questa modalità di sincronizzazione necessita la configurazione iniziale dei Android USB driver per Windows, se non sono già installati sul computer.

I driver USB per Windows sono specifici per ogni produttore del dispositivo Android, e sono disponibili gratuitamente dai siti web dei produttori. Il computer potrebbe già avere i driver installati se qualche software di sincronizzazione, backup o simili che si collega al vostro dispositivo tramite USB è stata installata precedentemente.

Controlla questo elenco per opzioni di scaricare e installare i Android USB drivers.

Android Sync - cavo USB trasferimento a Archivio

Sincronizzazione a Archivio di massa USB funziona solo su dispositivi Android con una scheda SD o una partizione equivalente. Questo metodo è inoltre più lenta da utilizzare in quanto richiede attivare e disattivare l'Archivio di massa USB ogni volta una sincronizzazione e avviata e durante la sincronizzazione.

Archivio USB è usato da telefoni e tablet Android per trasferire file con un computer tramite il cavo USB. Archivio USB è accessibile dopo che il cavo USB sia collegato, di solito nella barra di notifica superiore o la schermata Impostazioni. Questa modalità deve essere attivata e disattivata durante il trasferimento di note perché il computer e il dispositivo non può accedere alla scheda SD, allo stesso tempo.

Procedura per sincronizzare le note nuove dal dispositivo Android al computer:

  • collega il cavo USB
  • disattivare Archivio USB
  • premi il pulsante "Metti le note sul computer" sul dispositivo Android
  • attivare Archivio USB
  • clicca il pulsante "Ricevi le note nuove dal dispozitivo" sul computer software

Procedura per sincronizzare le note nuove dal computer al dispositivo Android:

  • collega il cavo USB
  • attivare Archivio USB
  • clicca il pulsante "Metti le note nuove sul dispozitivo" sul computer
  • disattivare Archivio USB
  • premi il pulsante "Ricevi nuove note dal computer" sul dispositivo Android

NOTA: solo le nuove note saranno trasferiti e aggiunti a quelli già esistenti, indipendentemente su ogni dispositivo che partecipa nella sincronizzazione. Per eliminare una nota definitivamente deve essere manualmente rimossa dal tutti i dispositivi che partecipano alla sincronizzazione.

BlackBerry Sync - cavo USB

Il BlackBerry può essere collegato a un computer con un cavo USB. Una volta collegato, verrà riconosciuto come un disco esterno al computer. Il computer può utilizzare questo disco per trasferire le nuove note con il tablet.

Il disco Blackberry richiede l'impostazione iniziale del software di supporto che viene fornito con il dispositivo. Questo software viene installato automaticamente quando il BlackBerry è collegato al computer per la prima volta. Alternativamente, può essere scaricato dal sito di supporto BlackBerry.

Procedura per sincronizzare le note nuove dal BlackBerry al computer:

  • collega il cavo USB e assicurarsi che l'unità BlackBerry sta mostrando in Windows Explorer
  • premi il pulsante "Metti le note sul computer" sul app di BlackBerry
  • clicca il pulsante "Ricevi le note nuove dal BlackBerry" sul computer software

Procedura per sincronizzare le note nuove dal computer al BlackBerry:

  • collega il cavo USB e assicurarsi che l'unità BlackBerry sta mostrando in Windows Explorer
  • clicca il pulsante "Metti le note nuove sul BlackBerry" sul computer software
  • premi il pulsante "Ricevi le note dal computer" sul app di BlackBerry

NOTA: solo le nuove note saranno trasferiti e aggiunti a quelli già esistenti, indipendentemente su ogni dispositivo che partecipa nella sincronizzazione. Per eliminare una nota definitivamente deve essere manualmente rimossa dal tutti i dispositivi che partecipano alla sincronizzazione.
