Baixar e instalar o controlador USB para Android

O Sincronização direta com o controlador USB necessita a configuração inicial dos controladores USB Android para Windows, se eles ainda não estiverem instalados no seu computador.

Os controladores USB para Windows são específicos para cada fabricante de dispositivos Android. Elas estão disponíveis gratuitamente nos sites de fabricantes. Provavelmente, o seu computador já tenha os controladores instalados.

Os controladores USB Android também são conhecidos como ADB (Android Debug Bridge).

O dispositivo Android deve ter o modo de depuração USB ativado antes de instalar os drivers e antes de usar Sincronização direta com o controlador USB:
 
Em dispositivos Android 1.x-3.x vá para Configurações > Aplicativos > Desenvolvimento > USB depuração ativado
 
Em dispositivos Android 4.0 e 4.1 vá para Configurações do sistema > Opções do desenvolvedor > Ativar depuração USB
 
Em dispositivos Android 4.2 e mais novos vá para Configurações do Sistema > Geral (Mais) > Opções do desenvolvedor > USB depuração ativado
 
NOTA: Em dispositivos Android 4.2 e mais novos (i.e Samsung Galaxy Note 3 e S4), pode ser necessário de desbloquear Opções do desenvolvedor antes de estar disponível no menu:
- prima "Configurações do sistema"
- prima "Sobre o dispositivo"
- prima o botão "Número de compilação" 7 vezes.
Modo de desenvolvedor agora deve ser desbloqueado e disponível em Configurações do Sistema > Geral (Mais) > Opções do desenvolvedor
