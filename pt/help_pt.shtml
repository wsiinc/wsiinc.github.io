Sincronização para Android

Diário Notas Multimédia irá sincronizar com telefones e tablet Android através rede sem fio (Wi-Fi) ou cabo USB. Sem acesso à Internet é necessária.

A sincronização Wi-Fi funciona quando o computador eo dispositivo Android são ligado à mesma rede sem fio (para o mesmo roteador de rede).

Sincronização via cabo USB pode ser realizado de duas maneiras: Sincronização direta com o controlador USB e Sincronização a armazenamento USB.

O Sincronização direta com o controlador USB é mais fácil de usar, mas ele exige a configuração inicial dos controladores USB Android para Windows, se eles já não estiverem instalados no seu computador.

O Sincronização a armazenamento USB não necessita de qualquer configuração, mas ele só funciona em dispositivos Android com um cartão SD ou uma partição equivalente. Este método também é mais lenta para usar porque requer o armazenamento USB a ser ligado e desligado cada vez que uma sincronização é iniciado e durante a sincronização.

Android sync - rede local sem fio (Wi-Fi)

Conecte o computador eo dispositivo Android na mesma rede sem fio (Wi-Fi).

O sincronização é iniciada abrindo a tela de sincronização Wi-Fi (a partir do menu Opções) no dispositivo Android e no menu correspondente no computador. A conexão será sempre iniciada PRIMEIRO a partir do telefone/tablet.

Caso seja desligado, prima o botão RECONECTAR no Android ANTES de clicar no botão CONECTAR no computador para restabelecer a conexão.

Depois que a conexão WiFi é estabelecida, clique no botão "Iniciar Sincronização" no computador para sincronizar as notas.

NOTA: selecionando a opção "Remover o todas as notas excluídos" irá remover permanentemente as notas apagados de todos os dispositivos e computadores que participam da sincronização.

Android sync através do cabo USB - controlador direto

O cabo USB deve ser conectado entre o computador eo dispositivo.

O dispositivo Android devem ter o modo de depuração USB ativado antes de usar Sincronização direta com o controlador USB:

Em dispositivos Android 1.x para 3.x vá para Configurações do systema - Aplicativos - Desenvolvimento - ativar a Depuração USB
 
Em dispositivos Android 4.0 e 4.1 vá para Configurações do sistema > Opções do desenvolvedor > Ativar depuração USB
 
Em dispositivos Android 4.2 e mais novos vá para Configurações do Sistema > Geral (Mais) > Opções do desenvolvedor > USB depuração ativado
 
Em dispositivos Android 4.2 e mais novos (i.e Samsung Galaxy Note 3 e S4), pode ser necessário de desbloquear Opções do desenvolvedor antes de estar disponível no menu:
- prima "Configurações do sistema"
- prima "Sobre o dispositivo"
- prima o botão "Número de compilação" 7 vezes.
Modo de desenvolvedor agora deve ser desbloqueado e disponível em Configurações do Sistema > Geral (Mais) > Opções do desenvolvedor

Este modo de sincronização é iniciado pela abertura da tela Sincronização direta com o controlador USB no dispositivo e no menu correspondente no computador. A conexão sempre será iniciado a partir do dispositivo PRIMEIRAMENTE.

Prima o botão CONECTAR NO DISPOSITIVO antes de clicar no botão Conectar no computador para estabelecer a conexão.

Quando a conexão USB está aberto, clique no botão "Iniciar Sincronização" no computador para sincronizar as notas.

NOTA: selecionando a opção "Remover o todas as notas excluídos" irá remover permanentemente as notas apagados de todos os dispositivos e computadores que participam da sincronização.

Sincronização direta com o controlador USB - configuração inicial

Este modo de sincronização requer a configuração inicial dos drivers USB Android para Windows, se eles já não estiverem instalados no seu computador.

Os controladores USB para Windows são específicos para cada fabricante de dispositivos Android. Eles são gratuitos a partir de sites de fabricantes. Provavelmente, o seu computador tem os controladores já instalados.

Verifique esta lista para opções do baixar e instalar controladores USB para Android.

Android sync através do cabo - armazenamento USB

Sincronização a armazenamento USB funciona somente em dispositivos Android com um cartão SD ou uma partição equivalente. Este método é mais lento para usar porque requer o armazenamento USB para ser ligado e desligado cada vez que uma de sincronização é iniciado e durante a sincronização.

O modo de armazenamento USB é usado em telefones e tablets Android para transferir arquivos com um computador via cabo USB. A configuração de armazenamento USB é acessível depois que o cabo USB está ligado, geralmente na barra de notificações superior ou a tela Configurações. Este modo deve ser ligado e desligado durante a transferência de notas porque o computador eo dispositivo não pode acessar o cartão SD simultaneamente.

Etapas para sincronizar novas notas a partir do dispositivo Android ao computador:

  • conecte o cabo USB
  • desativar o modo de armazenamento USB
  • prima o botão "Colocar novas notas para computador" na aplicativo de dispositivo
  • ativar o modo de armazenamento USB
  • clique no botão "Obtenha notas de Android" no software de computador

Etapas para sincronizar novas notas a partir do computador ao dispositivo Android:

  • conecte o cabo USB
  • ativar o modo de armazenamento USB
  • clique no botão "Colocar notas para Android" no software de computador
  • desativar o modo de armazenamento USB
  • prima o botão "Obtenha notas de computador" no aplicativo Android

NOTA: somente as novas notas serão transferidas e adicionadas aos que já existem, independentemente em cada dispositivo participante no sincronização. Notas existentes não serão atualizados pelo sincronização. Para excluir uma nota permanentemente lo deve ser removido manualmente a partir de todos os dispositivos participantes na sincronização.

BlackBerry sync - cabo USB

O BlackBerry pode ser conectado a um computador com um cabo USB. Uma vez conectado, irá mostrar como uma unidade externa ao computador. O computador irá utilizar esta unidade para transferir as novas notas com o BlackBerry.

A unidade Blackberry necessita a configuração inicial do software de suporte que vem com o dispositivo. Este software é geralmente instalado automaticamente quando o BlackBerry está conectado ao computador pela primeira vez. Alternativamente, ela pode ser baixada a partir do site de suporte do BlackBerry.

Etapas para sincronizar novas notas a partir do BlackBerry ao computador:

  • conecte o cabo USB e verifique se a unidade BlackBerry está mostrando no Explorador do Windows
  • prima o botão "Colocar notas para computador" no aplicativo BlackBerry
  • clique no botão "Obtenha notas de BlackBerry" no software de computador

Etapas para sincronizar novas notas a partir do computador ao BlackBerry:

  • conecte o cabo USB e verifique se a unidade BlackBerry está mostrando no Explorador do Windows
  • clique no botão "Colocar notas para BlackBerry" no software de computador
  • prima o botão "Obtenha notas de computador" no aplicativo BlackBerry

NOTA: somente as novas notas serão transferidas e adicionadas aos que já existem, independentemente em cada dispositivo participante no sincronização. Notas existentes não serão atualizados pelo sincronização. Para excluir uma nota permanentemente lo deve ser removido manualmente a partir de todos os dispositivos participantes na sincronização.
